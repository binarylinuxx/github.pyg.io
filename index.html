pyG

Python-scriptable desktop widget framework.

~/.config/pyg/config.py | style.scss

Features

  • Scriptable widgets (Bash, Python, Rust)
  • Dynamic content updates
  • Customizable layouts

Button Widget

Interactive elements that execute scripts on click.

widget.button.[button_name] [ child="dynamic-label/label_name" onclick-event="command_or_script_path" orientation="v|h" ]
onclick-event Path to script (Bash/Python/Rust) or direct command
Example:
widget.button.wifi_toggle [ child="dynamic-label.wifi_status" onclick-event="~/scripts/toggle_wifi.sh" orientation="h" ]

Label Widgets

Static Label

label or static label mean that simply static and never changeable widget.label.[label_name] [ text="Your text here" orientation="v|h" ]

Dynamic Label

dynamic label fetch output if for example enter simple command "date" the label will be output of the command or if you have more complex stuff like scripts feel free they work with dynamic labels widget.dynamic-label.[label_name] [ script-path="path_to_script/command" update-timing="seconds" #time per seconds for update ]

Window Widget

Container with flexible positioning.

widget.window.[window_name] [ anchor=["top","left","right","bottom"] child="widget_names" exclusivity="exclusive" ]

WidgetBox

Container for organizing multiple widgets.

widget.widgetBox.[box_name] [ child="widget_names" orientation="v|h" ]
